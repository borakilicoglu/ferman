# ferman v0.1.2

## Highlights

- Added a stronger JSON contract with stable success and error shapes
- Added stable machine-readable error codes
- Added multi-port support with batch summaries
- Added `--common`, `--plan`, `--doctor`, `--json-schema`, and `--watch`
- Expanded README and VitePress docs to cover the AI-facing CLI surface

## Notes
